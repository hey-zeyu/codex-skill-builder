# Example Prompts

Use these prompts to test or demonstrate `codex-skill-builder`.

## Build From A Rough Workflow

```text
Use codex-skill-builder. I repeatedly turn messy meeting notes into Jira tickets. Help me make that a Codex skill.
```

```text
Use codex-skill-builder. I want a skill that summarizes Douyin videos from links, screenshots, or pasted subtitles.
```

```text
Use codex-skill-builder. I have a weekly release checklist and want Codex to execute it consistently.
```

## Convert Documentation

```text
Use codex-skill-builder. Convert this CLI README into a skill that can safely operate the tool:
<paste README or provide path>
```

```text
Use codex-skill-builder. Turn this API documentation into a Codex skill with safety boundaries around credentials, rate limits, and external writes.
```

## Review Existing Skills

```text
Use codex-skill-builder. Review this skill folder for Codex fit, trigger quality, structure, and safety:
D:\path\to\my-skill
```

```text
Use codex-skill-builder. This skill works sometimes but does not trigger reliably. Help me improve its description and structure.
```

## Package For Sharing

```text
Use codex-skill-builder. Prepare this skill for GitHub release with a clean README, examples, validation command, and launch copy.
```

```text
Use codex-skill-builder. Package this skill into a distributable `.skill` archive after validation.
```
